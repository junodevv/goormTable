# GoormTable
식당 대기 어플

## 기획
캐치테이블, 테이블링과 같은 식당 대기 어플을 만들어보자!

## 요구사항 분석
로그인(관리자, 식당사장)
- 아이디와 비밀번호를 통해 로그인 할 수 있다.
  - 로그인시 예약 등록 화면으로 이동한다.

예약 등록
- 전화번호, 인원수, 추가 요청사항을 입렫받는다.
- 현재 대기중인 인원을 표시해준다.

예약자 명단 조회(관리자)
- 대기번호, 전화번호, 인원수, 추가 요청사항, 예약한 시간, 대기상태를 리스트로 확인할 수 있다.
  - 대기상태는 wait, confirm, call 로 나뉜다.
